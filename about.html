About Trade Manager

Overview

Trade Manager is a web-based tool for stock and derivatives traders, providing instant calculations for risk, reward, trade levels, and brokerage charges across major Indian brokers (Fyers, Upstox, Zerodha). The tool is designed to save time, reduce manual mistakes, and aid thoughtful trade planning whether for intraday or delivery positions.

Features

  • Calculate net P&L and trade levels for long/short trades.
  • Brokerage and statutory charges breakdown for compliance.
  • Supports Intraday and Delivery modes.
  • User can input stoploss or desired risk amount to compute targets and position size.
  • Basket function to organize multiple trades at once.
  • Minimal, touch-friendly design for quick use.

Usage

  1. Select broker and execution type (Intraday/Delivery).
  2. Input your position (Long/Short), entry, stoploss, exit, and quantity or risk.
  3. Press Calculate to view trade targets, breakdown and basket summary.
  4. Optionally, save or import trades for later review.

Technologies

  • HTML5, CSS, JavaScript — pure client-side
  • Optimized for fast use in all modern browsers
